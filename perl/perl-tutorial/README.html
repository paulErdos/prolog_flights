Perl Tutorial.

*****************************************************************

Perl is a scripting language.  The general property of a
scripting language is that it is very easy to get simple things
done very quickly.  Code can get to be unreadable, but it is
generally very brief and easy to write.  It is also useful for
throwaway programs (programs that are written, used once, and
discarded) and for simple formatting by means of pipes.

*****************************************************************

This file appears in two formats:  text and html.  In the html
version, the links are active, so if you are looking at this
file through a browser, choose the html version.

Some example programs are in the current directory:

hello.perl - standard intro to any language
argv.perl - command line arguments and progname
env.perl - prints out environment variables
errno.perl - prints out system error codes
ncat.perl - cat file: line numbers, no comments
getopts.perl - illustration of command line flags
cat.perl - cat(1) in perl with cmd line options
wc.perl - wc(1) with command line options
squeeze.perl - squeeze out multiple empty lines
xref.perl - cross reference utility
loaddict.perl - load dictionary into hash
text2html.perl - convert text to html
isprime.perl - check for prime numbers
rot13.perl - rotate 13 Caesar cypher
yes.perl - repeatedly agree to anything
merriam.perl - look up words in the M-W online dictionary
Lingua-Romana - a Greek algorithm in Latin

See the directories Examples1 and Examples2
